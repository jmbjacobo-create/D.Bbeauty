# D.B.Beauty Landing Page

Landing page estática para D.B.Beauty, preparada para subir a GitHub Pages.

## Archivos principales

- `index.html`: estructura de la página.
- `styles.css`: diseño responsive y estilos visuales.
- `script.js`: menú móvil y cabecera.
- `assets/optimized/deve-beauty-hero.jpg`: imagen principal de la landing.
- `assets/optimized/hidrafacial.jpg`: imagen para tratamientos hidrafaciales.
- `assets/optimized/carbon-peel.jpg`: imagen para carbón peel.
- `assets/optimized/unas-mirada.jpg`: imagen para uñas y mirada.
- `assets/optimized/fisioterapia.jpg`: imagen para fisioterapia.
- `assets/optimized/eliminacion-tatuajes.jpg`: imagen para eliminación de tatuajes.
